1️⃣ What is the difference between var, let, and const?
Ans: 
Var, let and const are the keyword of javascript for variable declaration.
where the var is the old method. can be reassign and redeclare. work in function scope
let use in block scope. can be reassign. it can not be redeclar.
const use for fixed value. it can not be reassign and redeclare.
